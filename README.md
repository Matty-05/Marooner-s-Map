### Step1: Installation

1. **Download XAMPP:** Go to [apachefriends.org](https://www.apachefriends.org/) and download the installer for your OS.
2. **Install:** Run the installer. Ensure that **Apache** and **MySQL** are checked in the component selection list.
3. **Launch:** Open the **XAMPP Control Panel** from your Start Menu or Applications folder.

### Step 2: Running the Services

1. **Start Apache:** Click the **Start** button next to Apache. The module name will turn green once it is active.
2. **Start MySQL:** Click the **Start** button next to MySQL. The module name will turn green once active.

### Step 3: Running the file in the server

### 1. Web Files (The Server)
Place your project folder or files in the `htdocs` directory:
* **Windows:** `C:\xampp\htdocs\`
* **macOS:** `/Applications/XAMPP/htdocs/`

To view your project, open your browser and go to: 
`http://localhost/your-folder-name/`

### 2. Running the website
Type: `http://localhost/your-folder-name/Marooner-s-Map/signup.html`
